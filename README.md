Kotlin Shooter MasterKotlinとSwingを使用して開発された、レトロスタイルの縦スクロールシューティングゲームです。滑らかな慣性移動、パーティクルによる爆発エフェクト、そして強力な弾幕を放つボスバトルを搭載しています。🚀 主な機能慣性移動システム: 物理ベースの摩擦係数を導入し、プレイヤー機の滑らかで快適な操作性を実現しています。ボス・エンカウント: スコアが一定に達すると、フェーズが切り替わり巨大なボスが出現します。弾幕アルゴリズム: ボスは全方位（360度）への放射状弾幕を放ち、プレイヤーに緊張感のある回避行動を促します。パーティクルエンジン: 敵機撃破時やプレイヤー被弾時に、動的に生成されたパーティクルが飛び散る視覚効果を実装しています。シーン管理: タイトル画面、プレイ中、ゲームオーバー、クリア画面のステートを列挙型（Scene）で管理しています。🕹 操作方法キーアクション← / →自機の左右移動（慣性が働きます）Spaceショットの連射 / ゲーム開始・リスタートESCゲームの終了🛠 技術スタックLanguage: KotlinFramework: Java Swing (JPanel, JFrame)Graphics: Graphics2D (アンチエイリアス有効)Loop Engine: javax.swing.Timer (60FPSターゲット)🏗 アーキテクチャ1. 衝突判定 (Collision Detection)java.awt.Rectangle を使用し、自機・敵機・弾のそれぞれの境界ボックスが交差するかを判定しています。2. 数学的な弾幕計算ボスの弾幕は、極座標系から直交座標系への変換を用いて計算されています。$$x = \cos(\theta) \times speed$$$$y = \sin(\theta) \times speed$$3. パーティクル・システム各パーティクルは独自の寿命（life）と速度ベクトルを持ち、フレームごとに減衰しながら描画されます。📂 セットアップ前提条件JDK 11 以上Kotlin コンパイラコンパイルと実行Bash# コンパイル
kotlinc GamePanel.kt -include-runtime -d Shooter.jar

# 実行
java -jar Shooter.jar
